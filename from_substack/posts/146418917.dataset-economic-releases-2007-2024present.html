I was working on something about whether the VIX does anything noteworthy in anticipation or after major macroeconomic releases, which meant I needed to finally clean up the previously mentioned economic releases dataset. Turned out it was incomplete, and also messy so I started from scratch. It’s all done now and I’m adding it to the supplementary material section along with the code I used to collect the data and clean it. Back to the regular schedule now.

The links to the data and code are beyond the paywall, but you can buy the data here for $39.

PS: The timestamps are in UTC, and the timestamp 20:59 represents ‘All Day’.

PSS: There’s a small issue with the ‘votes’ data from the MPC. Ignore any large values you find, they are from data conversion that didn’t work as expected and correcting it requires some regex. If I hack it I’ll update the dataset and let you all know. Apart from that everything else is fine.
